## Tech Stack
- Frontend: React, Tailwind CSS
- Backend: NestJS, Prisma
- Database: PostgreSQL (Railway)
- Auth: Session-based authentication

## Features
- User registration & login
- Create / edit / delete posts
- Comment system
- Protected routes

## Getting Started
npm install
npm run dev
